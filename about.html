StockFlow — Inventory Management System

StockFlow is a simple and professional inventory management website made using HTML, CSS and JavaScript. It helps manage item records, stock quantity, item status, assignments and reports with admin and visitor access.

Objective

  • To reduce manual inventory register work.
  • To manage item records digitally.
  • To provide admin and visitor level access.
  • To track quantity, status and item assignment.

Technology Used

  • HTML for webpage structure.
  • CSS for professional UI design.
  • JavaScript for functionality.
  • localStorage for browser data storage.

Main Modules

  • Login Module
  • Dashboard Module
  • Add Item Module
  • Inventory Records Module
  • Assign / Restock Module
  • Print Report Module

Features

  • Admin and visitor access.
  • Quantity tracking.
  • Low stock indicator.
  • Category overview cards.
  • Activity log.
  • Printable inventory report.

Future Scope

  • Backend authentication.
  • Database integration.
  • Supplier management.
  • Purchase and sales records.
  • PDF report generation.
