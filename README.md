# STOCKFLOW Web App.
[ Inventory Management System ]

Tech Stack:
- HTML
- CSS
- JavaScript
- localStorage

No react, no backend, no database setup, no terminal.

About Project:
Our project is StockFlow Web application, an Inventory Management System made using HTML, CSS and JavaScript.
It has role-based access where admin can manage inventory and visitors can only view records.
It includes quantity tracking, low stock indicator, category cards, activity log and printable reports.
Data is stored using browser localStorage.
